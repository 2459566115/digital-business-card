## 作品简介
这是一个由我自主设计、基于 HTML+CSS 开发的现代风格个人数字名片网页，以简历式布局呈现个人核心信息，兼顾视觉美观与实用功能。整体开发以自主实现为主，仅在细节优化环节少量参考 AI 建议，最终可通过 GitHub Pages 公开访问。

## 体现“软件作品”的一点
本作品并非简单的静态文本页面，而是遵循Web开发“结构与样式分离”的工程化思想，具备响应式适配能力（兼容多终端），代码包含交互微效果（标签hover、卡片阴影变化等），文件命名/结构规范、可维护性强，符合软件作品的基本特征（而非仅展示内容的静态页面）。

4. AI主要帮助的部分
针对 CSS 个别语法细节提供快速解答，节省查阅基础文档的时间；
提供现代UI设计思路（渐变、磨砂玻璃、微交互），提升页面视觉效果；

5. 对AI结果做的修改
调整CSS配色和交互效果（如标签hover颜色、头像尺寸），适配个人审美；
精简AI生成的语法细节，优化页面加载效率；
补充教育背景模块，完善个人标签（增加追剧、JavaScript等个性化内容）。

6. AI不可靠的一点
AI 给出的基础语法示例存在格式冗余问题，需要手动检查修复；
AI对“GitHub Pages开启步骤”的描述存在版本偏差，需结合实际GitHub界面验证；
AI生成的自我介绍偏模板化，缺乏个人特色，需完全替换为真实表述。
